# ThuLogin
python scripts to login to net.tsinghua.edu.cn
    
# Examples

1. Login to the net.tsinghua.edu.cn
    
    `python in.py your_account your_passwd net`
    
2. Logout the net.tsinghua.edu.cn 

    `python out.py`
